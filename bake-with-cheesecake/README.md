# Bake with Cheesecake!

Welcome to "Bake with Cheesecake!", a fun and interactive mini game where you can unleash your inner pastry chef! In this game, you'll embark on a delightful journey of baking various pastries while competing with friends on the leaderboard.

## Project Structure

The project is organized as follows:

```
bake-with-cheesecake
├── src
│   ├── __init__.py          # Initializes the src package
│   ├── main.py              # Entry point of the game
│   ├── leaderboard.py        # Manages leaderboard functionality
│   ├── user_interactions.py  # Handles user interactions
│   ├── commands
│   │   ├── __init__.py      # Initializes the commands package
│   │   ├── bake_pastries.py  # Logic for baking pastries
│   │   └── other_commands.py  # Additional game commands
├── requirements.txt          # Lists project dependencies
└── README.md                 # Project documentation
```

## Gameplay

In "Bake with Cheesecake!", players can:

- **Bake Pastries**: Use the `$bake` command to start baking delicious pastries. Follow the prompts to gather ingredients and complete the baking process.
- **Compete on the Leaderboard**: Track your scores and see how you rank against other players. The leaderboard is updated in real-time.
- **Interact with Cheesecake**: Engage with the game's character, Cheesecake, through various user interactions that enhance the gameplay experience.

## Setup Instructions

To get started with the game, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd bake-with-cheesecake
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
4. Run the game:
   ```
   python src/main.py
   ```
